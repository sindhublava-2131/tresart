# Structure: TresArt

## Root Directory
- `/frontend`: Frontend source code.
- `/backend`: Backend source code.
- `/logo`: Branding assets (logo.jpg).
- `/.planning`: Project management and research.

## Frontend Structure
- `/src/components`: Reusable UI components (Navbar, Hero, ProductCard, etc.)
- `/src/assets`: Static assets.
- `/public`: Public assets (icons, sounds).

## Backend Structure
- `index.js`: Main entry point and API definitions.
