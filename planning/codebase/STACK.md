# Tech Stack: TresArt

## Frontend
- **Framework**: React 19 (via Vite)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **HTTP Client**: Axios
- **Audio**: `use-sound`

## Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Middleware**: `cors`, `dotenv`
- **Data Management**: In-memory mock data (currently)

## Development
- **Package Manager**: npm
- **Linter**: ESLint
