# Conventions: TresArt

## JavaScript
- **Standard**: ES Modules (ESM) in both client and server.
- **Naming**: PascalCase for components, camelCase for variables and functions.

## Styling
- **Tailwind**: Utility-first styling with custom configurations in `tailwind.config.js` (or via CSS variables in Tailwind 4).
- **Aesthetics**: High-end, luxury, minimalist focus.

## Git
- **Commits**: Clear, descriptive messages.
- **Workflow**: Feature-based development.
